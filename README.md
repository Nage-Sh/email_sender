# Email Sender Automator

## Description
Hello,
In this program you could send automated accepted or rejected emails for startups and small scale business needs 
You just need to add the emails file in the program
Add your gmail addr and the app password

## Installation
You need the json, smtplib and email library

git clone https://github.com/Nage-Sh/email_sender.git
cd email_sender.git
python email_automator.py
